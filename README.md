MyCafeRecipe는 자기가 즐겨마시는 커스텀 음료 레시피를 공유하는 커뮤니티 사이트입니다.

기능
- 크게 회원가입, 게시판, 카페별 상품목록, 내 주변 카페찾기 기능을 구현
- 스타벅스, 할리스, 이디야 3가지 카페
- JAVA 및 JSP에서 해당 카페이름을 받아서 분류하여 화면단에 출력하는 동적 웹페이지로 구현

- 게시판
  1. 글 작성 시 파일업로드 AJAX로 비동기 구현
  2. 모달기능으로 상품목록에 있는 상품을 클릭하여 해당 상품정보가 레시피에 추가되도록 구현
  3. 레시피 추가와 전체삭제기능 (javascript)
  4. 답글달기 기능
  5. 페이징처리
  6. 검색기능
  
- 상품목록
  1. CRUD기능
  2. 수정 및 삭제는 로그인 시에만 가능
  3. 상품클릭 시 상품상세페이지로 이동
  
- 내 주변 카페찾기
  1. 카카오맵 OPEN API를 활용하여 본인의 주변 카페를 검색할 수 있게 함
